# Stellar Burgers Automated Tests

Проект автоматизированного тестирования для веб-приложения Stellar Burgers.

## Структура проекта
├── conftest.py # Фикстуры Pytest
├── locators.py # Локаторы элементов
├── test_login.py # Тесты авторизации
├── test_registration.py # Тесты регистрации
├── test_transition_and_constructor.py # Тесты переходов и конструктора
├── .gitignore # Игнорируемые файлы
└── README.md # Документация

## Фикстуры
driver - управление браузером
generate_name - генерация тестовых имен
generate_email - генерация email
generate_password - генерация паролей
registered_user - предварительная регистрация пользователя
logged_in_user - предварительный вход пользователя

## Тестовые сценарии
Регистрация нового пользователя
Авторизация различными способами
Переходы между разделами
Работа с конструктором бургеров
Выход из системы
